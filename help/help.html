How to use the QUDT Unit Conversion Tool


This tool is provided to convert from one unit type in the QUDT vocabulary to another. Like other online tools, you provide a value for the starting unit type and the tool provides the converted value in the target unit. Unlike other tools, this one has access to the entire QUDT quantity kind and unit vocabularies.

Usage

  1. You can start typing the name of a desired Quantity Kind and those matching what you the will show up in the Quantity Kind dropdown list. Or you can select a desired Quantity Kind from the list directly. The list is populated by executing a SPARQL query on the QUDT triple store, satisfying: ?quantitykind a qudt:QuantityKind (plus a filter against the text you have typed). Once a quantity kind is selected, this will narrow the types of starting and target units available to those that are applicable to the quantity kind. That is, they satisfy the following property in the QUDT triple store: qudt:applicableUnit ?unit. 
  2. The starting and target unit dropdowns are identical except that the target unit dropdown is not allowed to include the start unit value. Again, in SPARQL we apply a filter: FILTER (?tgtUnit != startUnit). 
  3. At any time, the start and target units are used to query for their associated conversion multipliers: start_unit qudt:conversionMultiplier ?stunit_cm. Initially, when no unit has been selected, the default units will be the first ones in the list. 
  4. Finally, the conversion is calculated as: converted_value = starting_value * starting_unit_conversion_multiplier / target_unit_conversion_multiplier.
